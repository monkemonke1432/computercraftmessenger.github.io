How to Download

There are two types of the messenger:

Client and Server.

-----

The client software connects to the server.

The server stands between client(s) so that they can communicate.

To install either, run either of these commands:

Client: wget https://raw.githubusercontent.com/monkemonke1432/ccm_server-client/refs/heads/main/client.lua

Server: wget https://raw.githubusercontent.com/monkemonke1432/ccm_server-client/refs/heads/main/server.lua

For those worried that this is malware; It isn't! Look at it for yourself, all the command does is downloads the file.

The program can only use one modem at a time, therefore you must have 2 separate servers (Wireless/Ender, Wired) if you want to.
